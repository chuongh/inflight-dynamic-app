# VietJet Ops Portal (Web)

Ground equipment operations portal — trolley carts, POS, iPad fleet tracking and repair workflows.

## Stack

- React 19 + TypeScript + Vite
- Ant Design 6 + Tailwind CSS 4
- TanStack Query (mock data layer, API-ready)

## Scripts

```bash
npm run dev          # local dev server
npm run build        # production build
npm run lint         # oxlint
npm run seed:mock-data
```
