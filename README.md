# Inflight Dynamic App — Prototype

Bản prototype thử nghiệm (không phải bản chính thức) của ứng dụng vận hành nhân viên.

## GitHub Pages

| Site | URL |
| --- | --- |
| Mobile (`index.html`) | https://chuongh.github.io/inflight-dynamic-app/ |
| Web portal (`portal-web`) | https://chuongh.github.io/inflight-dynamic-app/portal-web/ |

Deploy tự động qua GitHub Actions (`.github/workflows/deploy-pages.yml`) khi push lên `master`, hoặc chạy tay bằng **Actions → Deploy to GitHub Pages → Run workflow**.

**Cấu hình repo (một lần):** Settings → Pages → Source = **GitHub Actions** (không dùng branch `gh-pages` nữa).
